# 🎓 Student Performance Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze student academic performance, attendance, grades, and department-wise outcomes. It helps identify performance trends and supports data-driven educational insights.

---

## 🎯 Objectives
- Monitor student academic performance
- Analyze attendance patterns
- Compare department-wise performance
- Track pass and fail rates
- Evaluate grade distribution

---

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

---

## 📊 Dashboard Features
- Total Students KPI
- Average Attendance KPI
- Average Total Marks KPI
- Pass Percentage KPI
- Department-wise Student Analysis
- Student-wise Total Marks
- Grade Distribution
- Pass vs Fail Analysis
- Attendance and Marks Relationship
- Gender-wise Performance Analysis
- Interactive Filters and Slicers

---

## 📈 Key Insights
- Identified top-performing departments
- Analyzed attendance impact on marks
- Compared student performance across departments
- Evaluated grade and pass percentage distribution

---

## 🚀 Outcome
The dashboard provides a clear view of student performance metrics and helps educators make informed decisions through interactive visualizations.
